# 🔤 Word & Text Counter

A simple and responsive **Word & Text Counter** built with **Vanilla JavaScript**. It updates the word count and character count in real time while typing and provides visual feedback when the maximum character limit is exceeded.

🔗 **Live Demo:** https://haseem18.github.io/JS-Mini-Projects/07-Word-&-Text-Counter/

---

## ✨ Features

- ⌨️ Real-time word counting
- 🔤 Live character counter (excluding leading and trailing spaces)
- 📏 Maximum character limit of **150**
- 🚨 Visual warning when the limit is exceeded
- 🧹 Ignores leading and trailing spaces using `trim()`
- 📱 Fully responsive design

---

## 🛠️ Tech Stack

- **HTML5** — Semantic structure
- **CSS3** — Responsive layout, custom styling, visual feedback
- **Vanilla JavaScript** — DOM Manipulation, Event Handling, String Processing

---

## 🧠 What I Learned

- Listening to user input with the `input` event
- Counting words using `split()` and `filter()`
- Using `trim()` to ignore leading and trailing spaces
- Dynamically updating the DOM as the user types
- Toggling CSS classes with `classList.add()` and `classList.remove()`
- Providing visual feedback based on application state
- Building a responsive UI for text-based applications
