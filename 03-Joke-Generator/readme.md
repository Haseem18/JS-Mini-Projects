# 😂 Joke Generator

A fun and interactive web app that displays random Hinglish jokes along with a character name and profile image every time the user clicks the **Next Joke** button.

🔗 **Live Demo:** https://haseem18.github.io/JS-Mini-Projects/03-Joke-Generator/

---

## ✨ Features

- 😂 Displays random Hinglish jokes
- 👤 Shows character name with each joke
- 🖼️ Dynamic profile image update
- 🎲 Random joke generation on button click
- 🚀 Auto-loads first joke on page load
- 📱 Fully responsive design
- 🎨 Clean modern UI with dark theme

---

## 🛠️ Tech Stack

- **HTML5** — Semantic structure
- **CSS3** — Flexbox, responsive design, modern UI styling
- **JavaScript (Vanilla)** — Arrays, Objects, DOM manipulation, Event Listeners, Randomization

---

## 🧠 What I Learned

- Working with arrays of objects
- Random selection using `Math.random()` and `Math.floor()`
- Dynamically updating DOM elements (`textContent`, `src`)
- Handling user events with `addEventListener`
- Structuring reusable functions for cleaner code
- Building interactive UI without any libraries
