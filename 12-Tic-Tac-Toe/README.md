# ❌ Tic Tac Toe

A simple and interactive Tic Tac Toe game built using **HTML, CSS, and Vanilla JavaScript**.

This project was created as part of the **Thunder Batch (100 Days of Code)** challenge by **Rohit Negi (Coder Army)** to practice DOM manipulation, event handling, arrays, and game logic.

## ✨ Features

* Two-player gameplay (X vs O)
* Real-time turn indicator
* Winner detection
* Draw detection

## 🎮 How to Play

1. Player **X** starts first.
2. Players take turns placing their mark on the board.
3. The first player to align three marks horizontally, vertically, or diagonally wins.
4. If all cells are filled and no winner is found, the game ends in a draw.
5. Click the **Reset** button to start a new game.
