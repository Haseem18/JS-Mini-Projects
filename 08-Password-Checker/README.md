# 🔐 Password Strength Checker

A clean and interactive password strength checker built with **Vanilla JavaScript**. It analyzes the entered password in real time and categorizes it as **Weak**, **Normal**, or **Strong** based on its length and character composition. It also includes a password visibility toggle for a better user experience.

🔗 **Live Demo:** https://haseem18.github.io/JS-Mini-Projects/08-Password-Checker/

---

## ✨ Features

- 🔒 Real-time password strength detection
- 👁️ Show / Hide password toggle
- 🟢 Categorizes passwords as **Weak**, **Normal**, or **Strong**
- 🔤 Detects lowercase and uppercase letters
- 🔢 Detects numeric digits
- ✨ Detects special characters
- 📏 Requires a minimum length of 8 characters for stronger passwords
- 🎨 Dynamic UI updates with different colors for each strength level
- 📱 Fully responsive design
