# ⏰ Digital Clock

A responsive **Digital Clock** built using **HTML, CSS, and Vanilla JavaScript**. The clock displays the current time and date in real time, updating every second using the JavaScript `Date` object.

🌐 **Live Demo:** https://haseem18.github.io/JS-Mini-Projects/09-Digital-Clock/

---

## ✨ Features

- ⏱️ Live digital clock updating every second
- 📅 Displays the current date
- 🌍 Uses `toLocaleTimeString()` and `toLocaleDateString()`
- 🕐 Supports locale-based time formatting
- 📱 Fully responsive design
- 🎨 Modern glassmorphism-inspired UI

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

---

## 📚 Concepts Practiced

- JavaScript `Date` Object
- `setInterval()`
- `toLocaleTimeString()`
- `toLocaleDateString()`
- DOM Manipulation
- Real-time UI Updates
- Responsive Design
