# 💬 Quote Generator

A clean and interactive web app that displays a random motivational quote along with the author's name and profile image every time the user clicks the **Generate New Quote** button.

🔗 **Live Demo:** https://haseem18.github.io/JS-Mini-Projects/02-Quote-Generator/

---

## ✨ Features

* 💬 Displays a random motivational quote
* 👤 Shows the corresponding author's name
* 🖼️ Dynamically updates the author's profile image
* 🎲 Random quote generation on every button click
* 🚀 Automatically displays a quote when the page loads
* 📱 Fully responsive layout (mobile-friendly)
* 🌙 Premium dark UI with elegant gold accents

---

## 🛠️ Tech Stack

* **HTML5** — Semantic structure
* **CSS3** — Flexbox, `clamp()` for responsive typography, custom dark theme
* **JavaScript (Vanilla)** — Arrays, Objects, DOM manipulation, Event Listeners, Randomization

---

## 🧠 What I Learned

* Storing structured data using an array of objects
* Selecting a random object using `Math.random()` and `Math.floor()`
* Updating multiple DOM elements dynamically
* Using object destructuring for cleaner function parameters
* Separating application logic into reusable functions
* Initializing the UI by calling functions on page load
* Building a clean, responsive card-based interface
