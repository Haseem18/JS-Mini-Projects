# 📝 To Do List App

A simple and responsive **To Do List App** built using **HTML, CSS, and Vanilla JavaScript** as part of my **Thunder Batch (100 Days of Code)** journey by **Rohit Negi (Coder Army)**.

This project focuses on DOM manipulation, event delegation, form handling, validation, and dynamic UI updates.

🌐 **Live Demo:** https://haseem18.github.io/JS-Mini-Projects/16-To-Do-List/

---

## 🚀 Features

✅ Add new tasks

✅ Edit existing tasks

✅ Delete tasks

✅ Prevent empty tasks

✅ Prevent duplicate tasks

✅ Maximum 10 tasks limit

✅ Error handling with user-friendly messages

✅ Responsive design

✅ Event Delegation for better performance

---

## 🛠️ Concepts Covered

* DOM Manipulation
* Forms & Input Handling
* Event Listeners
* Event Delegation
* Dynamic DOM Creation
* Arrays
* Array Methods (`push`, `splice`, `some`, `indexOf`)
* Form Validation
* Content Editable Elements

* Responsive Design
