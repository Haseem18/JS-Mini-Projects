# ❤️ Love Calculator

A fun and interactive **Love Calculator** built with **Vanilla JavaScript** that calculates a fictional love percentage based on two names. The project focuses on input validation, DOM manipulation, conditional logic, and dynamic UI updates.

🔗 **Live Demo:** https://haseem18.github.io/JS-Mini-Projects/06-Love-Calculator/

---

## ✨ Features

* ❤️ Calculates a fun love percentage
* 👦👧 Accepts two names as input
* ✅ Validates both input fields
* 🚫 Rejects empty inputs
* ❌ Prevents numbers, spaces, and special characters in names
* 💌 Displays a personalized compatibility message based on the result
* 🔔 Animated toast notifications for validation errors
* 📱 Fully responsive design
* 🎨 Modern pink-themed UI

---

## 🛠️ Tech Stack

* **HTML5** — Semantic structure and forms
* **CSS3** — Flexbox, Responsive design, Animations, Glassmorphism-inspired UI
* **JavaScript (Vanilla)** — DOM Manipulation, Event Handling, Input Validation, Conditional Logic

---

## 🧠 What I Learned

* Validating form inputs using JavaScript
* Preventing form submission with `event.preventDefault()`
* Using `trim()` to remove unwanted whitespace
* Checking every character using `Array.every()`
* Dynamically updating text with `textContent`
* Showing and hiding elements using CSS classes
* Creating reusable toast notifications with `createElement()`
* Using `requestAnimationFrame()` for smooth animations
* Handling multiple conditions using `if...else`
* Writing cleaner and reusable functions
