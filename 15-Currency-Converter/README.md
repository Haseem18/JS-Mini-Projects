# 🌍 Currency Converter

A modern and responsive Currency Converter built using **HTML, CSS, and Vanilla JavaScript**.

This project allows users to convert currencies in real-time using live exchange rates fetched from the **Frankfurter API**. It focuses on learning API integration, asynchronous JavaScript, form handling, and error management.

🌐 **Live Demo:** https://haseem18.github.io/JS-Mini-Projects/15-Currency-Converter/

---

## ✨ Features

* 💱 Convert between multiple currencies
* 🌍 Live exchange rates using Frankfurter API
* 🔄 Swap currencies with a single click
* ⚡ Fast conversion using Fetch API
* ⏳ Loading indicator while fetching data
* ❌ User-friendly error handling
* 📱 Fully responsive design
* 🎨 Modern UI with clean layout

## 🚀 How It Works

1. Enter an amount.
2. Select the source currency.
3. Select the target currency.
4. Click **Convert**.
5. The application fetches the latest exchange rate from the API.
6. The converted value and exchange rate are displayed instantly.

## 🔌 API Used

### Frankfurter API

```text
https://api.frankfurter.dev/
```

Example:

```text
https://api.frankfurter.dev/v1/latest?base=USD
```
