# 🔢 Counter

A clean and responsive counter application built with **Vanilla JavaScript**. Users can increase, decrease, and reset the counter while handling edge cases with animated toast notifications.

🔗 **Live Demo:** https://haseem18.github.io/JS-Mini-Projects/05-Counter/

---

## ✨ Features

- ➕ Increase the counter
- ➖ Decrease the counter
- 🔄 Reset the counter
- 🚫 Prevents the counter from going below **0**
- 🚫 Prevents the counter from exceeding **10**
- 🔔 Animated toast notifications for invalid actions
- 📱 Fully responsive design
- 🎨 Modern glassmorphism-inspired UI

---

## 🛠️ Tech Stack

- **HTML5** — Semantic structure
- **CSS3** — Flexbox, Glassmorphism, Responsive design, CSS Transitions
- **JavaScript (Vanilla)** — DOM manipulation, Event Listeners, Dynamic Element Creation

---

## 🧠 What I Learned

- Updating the DOM dynamically with `textContent`
- Handling click events using `addEventListener`
- Managing application state with variables
- Creating DOM elements dynamically using `document.createElement()`
- Inserting elements using `before()`
- Using `requestAnimationFrame()` for smooth UI animations
- Creating reusable functions (`renderCount`, `warnMsg`)
- Handling edge cases with conditional logic
- Removing dynamically created elements after animations using `setTimeout()`
