# 🏅 Olympic Countdown Timer

A responsive countdown timer built with **HTML, CSS, and Vanilla JavaScript** that displays the remaining **days, hours, minutes, and seconds** until the **Los Angeles 2028 Summer Olympics Opening Ceremony**.

🌐 **Live Demo:** https://haseem18.github.io/JS-Mini-Projects/10-Olympic-Countdown/

---

## ✨ Features

* ⏳ Live countdown updated every second
* 🏅 Countdown to the **Los Angeles 2028 Summer Olympics**
* 📅 Displays Days, Hours, Minutes, and Seconds remaining
* 🎨 Modern glassmorphism UI with Olympic-inspired colors
* 📱 Fully responsive design
* ⚡ Built using pure JavaScript (no libraries)

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* Vanilla JavaScript

---

## 📚 Concepts Practiced

* JavaScript Date Object
* `Date.now()`
* `setInterval()`
* Time Calculations
* DOM Manipulation
* Mathematical Operations (`Math.floor`)
* Responsive UI Design
