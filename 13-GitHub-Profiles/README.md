# 🐙 GitHub Profiles Explorer

A modern GitHub Profiles Explorer built using **HTML, CSS, and Vanilla JavaScript**.

This project fetches real GitHub user data using the **GitHub REST API** and displays profiles in a responsive card layout. Users can search the number of profiles to load and view detailed information about any GitHub account.

Built as part of the **Thunder Batch (100 Days of Code)** challenge by **Rohit Negi (Coder Army)**.

## 🌐 Live Demo

https://haseem18.github.io/JS-Mini-Projects/13-GitHub-Profiles/

---

## ✨ Features

* Fetch real GitHub users using GitHub API
* Responsive profile card layout
* Dynamic profile loading
* Debounced search input
* Detailed user profile page
* Followers, Following, Repositories, and Gists statistics
* Loading spinner while fetching data
* Single-page application style navigation
* Mobile responsive design
* Error handling and fallback values
