# ⏱️ Stopwatch

A responsive **Stopwatch** built using **HTML, CSS, and Vanilla JavaScript**. It allows users to **start, stop, resume, and reset** the timer while displaying the elapsed **hours, minutes, and seconds** in real time.

🌐 **Live Demo:** https://haseem18.github.io/JS-Mini-Projects/11-Stopwatch/

---

## ✨ Features

- ▶️ Start the stopwatch
- ⏸️ Stop/Pause the timer
- 🔄 Resume from where it stopped
- ♻️ Reset the timer to `00:00:00`
- ⏱️ Real-time updates every second
- 📱 Fully responsive UI
- ⚡ Built using pure JavaScript (no libraries)

---
